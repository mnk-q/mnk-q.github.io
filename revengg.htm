Reverse Engineering event is being conducted by PCC club, under Innovac 21. This is an unconventional coding contest, which will check your problem solving skill as well as cryptographic skills.

Participate and outsmart the other participants to reach at the peak of the Leaderboard. May the best coder win!

This will be a 3 Hour Coding contest.
You have to download the problem set from the Problems Tab

Problem set will be a zip file, containing 7 problems. Each problem set contains an exe file, an input.txt, output.txt and instruction.txt.
Executing the exe file will take input from input.txt and write output in output.txt

You will have to figure out the logic behind the executable file, and write code for it, in the contest. Your code will be checked against various test cases. The Problem names are encrypted, before submitting, you have to verify the challange name with problem name.

  • It is an individual event.
  • Doubts regarding anything should be asked directly to the event coordinators only. Posting it in public forums will not be entertained.
  • Unfair means to guarantee advantage will lead to disqualification.
  • The solutions should not be made public or open-sourced during the contest.
  • The decision of the organizers will be final and binding in any case of disputes.

Judging Criteria
The participant with the highest score wins.
